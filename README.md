# SLAM Vision - ROS2 with KITTI Dataset

A Visual SLAM implementation using ORB-SLAM3 in stereo mode with the KITTI odometry dataset, built on ROS2 Jazzy.

## Stack
- Ubuntu 24.04
- ROS2 Jazzy
- ORB-SLAM3 (Stereo mode)
- KITTI Odometry Dataset
- Pangolin (visualization)
